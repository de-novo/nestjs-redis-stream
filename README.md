## Installation

### with npm

```sh
npm i @de-novo/nestjs-redis-streams
```

### with yarn

```sh
yarn add @de-novo/nestjs-redis-streams
```

### with pnpm

```sh
pnpm i @de-novo/nestjs-redis-streams
```
